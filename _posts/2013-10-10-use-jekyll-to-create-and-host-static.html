What is jekyll?

According to the introduction of jekyll github project:

"Jekyll is a simple, blog aware, static site generator."

Github page are Jekyll recognizable, which means the static site created by Jekyll can automatically be built and hosted by github.

Today, I tried it and create a simple personal site on github minus1.se. Jekyll is pretty cool.
