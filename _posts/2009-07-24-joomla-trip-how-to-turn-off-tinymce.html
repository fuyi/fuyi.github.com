Today, I was editing a article in Joomla, I want to wrap text around a image, but I found this function doesn't work in Joomla.

My WYSWYG editor in joomla is the default one: TinyMCE Version: 3.2.4.1

the probelm is : after I update the html element with attribute style="float:left", which could implement text wrap effecct, this update is automatically stripped away by TinyMCE. Thus, the text wrap doesn't work.

I searched on web, the answer to this question is: by default, TinyMCE enables "HTML filter" function, it strips away any html element from its content which is against its predefined rules, like XHTML standard.

Obviously, in my case, "HTML filter" function is enabled, which causes the probelm I encountered. There are several ways to solve this headover, The simpleast way is to turn off this function and allow any html element/attributes.
