Most traffic is from search engine. How to make target customer find your website is the key to success. Google is the giant in web search world, However, Baidu beat Google in Chinese local market(over 70% market share). Therefore, for a website to succeed in China, it has to do search engine optimization (SEO) for Baidu.




As a competitor of Google, Baidu specilízes on Chinese search, it has many difference against Google in many aspects, from algorithem to strategy, although similarity is the case. So, it is prudent to study these differences and find the balance when seeking SEO solution.




In this article, I will discuss the similarity and differences as 3 stages from search engine´s perspective.




How does a search engine include a website into their search result. We can see it as 3 steps:


1) search engine find the website


2) search engine "learn" what this website is about: learn the content of the website


3) search engine index website (decide which position the site should be for a certain keyword)




In the following sections, we are going to discuss each step and compare the principle Google and Baidu employ.




1) search engine find the website




search engine use robot spider to crawl the whole web. the spider find new site by following anchor-links on the web. So, it takes some time (several days maybe) for the spider to find new site. This process is the same.




Or webmaster can actively submit website to search engine and let them know the site.




2) search engine "learn" what this website is about: learn the content of the website.




spider check through all the pages by following all the links, start from home page.




sitemap format:


Google: use sitemap.xml as standard format


Baidu: doesn't use so called "Baidu News Protocol” format




Robots.txt


Google: Google use robots.txt file to set rule , the Googlebot will follow the rule to crawl the site


Baidu: Just ignore robots.txt




3) search engine index website (decide which position the site should be for a certain keyword)
